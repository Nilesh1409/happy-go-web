🔧 Technical Changes Applied

1. Button Visibility Fixes:

  • Added inline styles to all modal buttons with explicit backgroundColor
  • Color fallbacks: Each button has a specific hex color that works without CSS variables
  • Disabled states: Added opacity: 0.5 and cursor: not-allowed for disabled buttons

2. Scrolling Improvements:

  • Enhanced DialogContent: Added webkitOverflowScrolling: "touch" for iOS
  • Better overflow handling: overflowY: "auto", overflowX: "hidden"
  • Improved max-height: Uses calc(100vh - 2rem) as fallback
  • Content padding: Ensured registration form content can expand properly

3. CSS Fallbacks Added:

  • .bg-green-600, .bg-blue-600, .bg-purple-600 - explicit background colors
  • .overflow-y-auto - enhanced scrolling with touch support
  • .max-h-[95vh] - better viewport height fallbacks
  • Hover state fallbacks for all button colors
